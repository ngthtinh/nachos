# nachos
## Compile Nachos
Vào thư mục NachOS 4.0/code/build.linux

Mở terminal tại thư mục này.

Gõ:

$ make depend

$ make

Test:

$ ./nachos


## Compile Ứng dụng người dùng

Vào thư mục NachOS 4.0/code/test

Mở terminal tại thư mục này.

Gõ:

$ make

Test:

$ ../build.linux/nachos -x <tên chương trình cần chạy>
